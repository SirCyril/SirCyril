Introduction¶

For my project report, I analysed a dataset obtained from the TMDB Movie Database (TMDb). This data contains details on approximately 10,000 movies produced from 1960 to 2015. The dataset contains 21 columns as shown below:

1. id - a unique random identifier used to identify each movie
2. imdb_id - a unique official alphanumeric identifier which is can be used to google each movie
3. popularity - a numeric value assigned to each movie signifying how popular the movie is
4. budget - dollar value attributing how much was spent to produce each movie 
5. revenue - dollar value attributing how much was earned in sales from each movie
6. original_title - signifies the official name used to identify each movie
7. cast - displays a list of the top 5 actors/actresses in each movie
8. homepage - contains a link to the movie website
9. director - indivual(s) responsible for directing the movie
10. tagline - a short description or comment unique to each movie
11. keywords - word(s) associated with the movie title and can be used to search for the movie 
12. overview - a short description on what each movie is about 
13. runtime - signifies the lenght of the movie in minutes
14. genres - categorization of what type of movie it is
15. production_companies - the name of the company associated with producing wach movie
16. release_date - date each movie was produced (in format mm/dd/yy)
17. vote_count - total votes each movie received used by users
18. vote_average - rating received by each movie based on average vote count
19. release_year - year each movie was produced
20. budget_adj - dollar value of budget in terms of 2010 dollars, accounting for inflation over time
21. revenue_adj - dollar value of revenue in terms of 2010 dollars, accounting for inflation over time

Questions for Analysis¶

1. Most popular movie in each release_year?
2. Highest revenue generated in each release year?
3. Is there a correlation between popularity and revenue?
4. Have movie length(runtime) increased over the years?

Conclusions¶

1. The highest grossing movie over the period under review was Avator with over 2.9B US Dollars
2. Popularity of a movie does not contribute significantly to revenue generated.
3. The longest movie over the period under review was 'The Story of Film: An Odyssey' which was 900 minutes 
4. Generally, movies lenghts have increased over the period under analysis

Limitations¶

  1. dataset for analysis had approximtely 10,000 null(NaN) fields thus required quite a bit of cleaning as well as improvisation to be able to achieve meaninful analysis.
  2. longest movie runtime of 900minutes is an outlier. This increases the error variance and results in bias of analysis of longest movie.
